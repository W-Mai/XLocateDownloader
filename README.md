# XLocate Downloader

## ＳＴＡＴＵＳ提示信息

|  状态       | 含义        |
|  ----     | ----         |
|   S:E     |   文件已存在                   |
|   S:F     |   文件保存失败                  |
|   C:数字    |   网络问题，数字为HTTP状态码     |
|   L:0     |   下载下来的文件长度为零         |
